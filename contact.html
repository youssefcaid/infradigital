Questions fréquentes

Comment accéder aux fichiers PDF ?

Tous les fichiers sont disponibles gratuitement sur les pages des modules (1ère année et 2ème année). Cliquez sur le bouton correspondant pour ouvrir ou télécharger le fichier.

Les ressources sont-elles vraiment gratuites ?

Oui, toutes les ressources pédagogiques sur ce site sont entièrement gratuites. Notre mission est de soutenir les étudiants OFPPT sans frais.

Comment signaler un problème avec un fichier ?

Si vous rencontrez un problème avec un fichier (lien brisé, erreur dans le contenu), contactez-nous par email avec le nom du module et la nature du problème.

Conseils pour une communication efficace

  • Mentionnez toujours le module concerné dans votre message
  • Soyez précis dans votre question ou problème
  • Pour WhatsApp, envoyez d'abord un message texte avant d'appeler
  • Respectez les horaires de disponibilité
